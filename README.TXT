IMPORTANT NOTE FOR MATLAB R13 (6.5) USERS

On Matlab r13 systems there is a bug in the COM-tools.  The hysyslib won't work
properly unless you patch your Matlab system.  For more info, see the link
to Mathworks support information below:

http://www.mathworks.com/support/solutions/data/1-1ABU3.html?solution=1-1ABU3
